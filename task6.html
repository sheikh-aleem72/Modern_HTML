Differences between HTML and HTML5

HTML HTML5
HTML stands for Hypertext Markup Language. HTML5 stands for Hypertext Markup Language version 5.
HTML is the older version of the markup language. HTML5 is the latest version of the markup language.
HTML does not support native audio and video playback. HTML5 provides native support for audio and video elements, allowing easy embedding of multimedia content.
HTML does not have built-in form validation. HTML5 introduces client-side form validation, reducing the need for JavaScript for basic form validation.
HTML does not have native support for canvas for graphics rendering. HTML5 introduces the <canvas> element for graphics, enabling dynamic graphics and animations without the need for plugins.
